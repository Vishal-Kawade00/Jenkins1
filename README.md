🚀 OS-Visualizer

OS-Visualizer is an interactive web application that visualizes core Operating System scheduling algorithms in real-time.

Built using React 19 + Vite, it provides animated process simulation, dynamic Gantt charts, and automatic performance metric calculations to help users understand how scheduling strategies impact system performance.

✨ Features

🔄 Interactive step-by-step & auto simulation

🧠 Supports FCFS, SJF (Preemptive/Non-Preemptive), Priority (Preemptive/Non-Preemptive), Round Robin

📊 Automatic calculation of:

Waiting Time

Turnaround Time

Response Time

CPU Utilization

Throughput

📈 Real-time Gantt Chart visualization

⚙️ Custom process input support

🛠 Tech Stack

Frontend: React 19, Vite

Styling: Tailwind CSS

Animations: Framer Motion

State Management: React Context API + useReducer

🚀 Getting Started
npm install
npm run dev
